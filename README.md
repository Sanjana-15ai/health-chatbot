# 🩺 Health Chatbot

A smart **AI-powered Health Chatbot** that helps users get basic health information, symptom guidance, and wellness suggestions through a conversational interface.

## 📌 Project Overview

The Health Chatbot is designed to provide users with quick and accessible health-related information. It can answer common health questions, provide symptom suggestions, and guide users toward healthy practices.

This chatbot is intended for **informational purposes only** and should not replace professional medical advice.

---

## 🚀 Features

* 💬 Interactive chatbot interface
* 🧠 Basic symptom checking
* 📋 Health tips and recommendations
* 🔍 Quick responses to common health questions
* 🌐 Simple and user-friendly UI

## ⚙️ Installation

1. Clone the repository

```
git clone https://github.com/Sanjana-15ai/health-chatbot.git
```

2. Navigate to the project directory

```
cd health-chatbot
```

3. Install dependencies

```
npm install
```

4. Start the application

```
npm start
```
